HTML Table Terminology

Table Elements

The table below contains all the HTML elements (or tags) you will ever need to know to be able to build a table.

Element Name Description
<table> Table Encloses all the information inside the table, including headers, data etc.
<thead> Table Head Encloses the row in which each column heading is written.
<th> Table Header Contains the heading for a particular column. Goes inside <thead>.
<tbody> Table Body Contains all the table data (excluding the column headings i.e. excluding everything you'd put in <thead>).
<tr> Table Row Contains everything you would want to include in one particular row
<td> Table Data Contains all the data you would want in one cell
<tfoot> Table Foot Confines the set of row(s) (usually the last row) that make up the footer of a table

Table Attributes

The following attributes are written inside the <td> element and dictate the number of cells a particular <td> element spans over.

Attribute Name Description
colspan Column Span Defines the number of columns a <td> (or one cell) will span.
rowspan Row Span Defines the number of rows a <td> (or one cell) will span.

 Example


<!DOCTYPE html>

<html>

<body>

<h2>Basic HTML Table</h2>


<table>

    <thead>

        <th>First name</th>

        <th>Last name</th>

        <th>Age</th>

    </thead>

    <tr>

        <td>Jill</td>

        <td>Smith</td>

        <td>50</td>

    </tr>

    <tr>

        <td>Eve</td>

        <td>Jackson</td>

        <td>50</td>

    </tr>

    <tr>

        <td>John</td>

        <td>Doe</td>

        <td>80</td>

    </tr>

</table>

</body>

</html>


The code above generates the following table:

Basic HTML Table
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
John Doe 80

Then adding the attributes colspan and rowspan to the same chunk of code adjusts the length and width of a particular cell. In this case, the length and width of two cells each containing a header.


<!DOCTYPE html>

<html>

<body>

<h2>Basic HTML Table</h2>


<table>

    <tr>

        <th colspan="2">Full Name</th>

        <th rowspan="2">Age</th>

    </tr>

    <tr>

        <th>First name</th>

        <th>Last name</th>

    </tr>

    <tr>

        <td>Jill</td>

        <td>Smith</td>

        <td>50</td>

    </tr>

    <tr>

        <td>Eve</td>

        <td>Jackson</td>

        <td>50</td>

    </tr>

    <tr>

        <td>John</td>

        <td>Doe</td>

        <td>80</td>

    </tr>

</table>

</body>

</html>



Basic HTML Table
Full name Age
First name Lastname
Jill Smith 50
Eve Jackson 94
John Doe 80

Note how the two <th> elements containing the text "Full Name" and "Age" have each extended over two rows and two columns in the table above.
